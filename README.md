# Initial steps towards a lab website

# Folder structure

/my-lab-website
│── index.html  (Home)
│── members.html  (Lab Members)
│── research.html  (Research)
│── contact.html  (Contact)
│── style.css  (CSS file)
│── images/  (Folder for member photos, graphical abstracts, etc.)


To run locally:
 python3 -m http.server 8000
 
 Then open 
http://localhost:8000
 in your browser
